# <a name="operator-mask"></a>MASK 演算子

返しますがビット マスク内のビット*recordfieldname*または*レコード*が設定し、その他のすべてのビットがクリアされます。

## <a name="syntax"></a>構文

> マスク {recordfieldname | レコード}
