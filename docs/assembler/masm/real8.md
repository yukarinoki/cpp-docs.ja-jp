# <a name="real8"></a>REAL8

Allocates and optionally initializes a double-precision (8-byte) floating-point number for each *initializer*.

## <a name="syntax"></a>構文

> *name* **REAL8** *initializer* ⟦ __,__ *initializer* ...⟧
