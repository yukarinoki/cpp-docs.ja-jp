# <a name="operator-dup"></a>DUP 演算子

Specifies *count* number of declarations of *initialvalue*.

## <a name="syntax"></a>構文

> *count* **DUP** __(__ *initialvalue* ⟦ __,__ *initialvalue* ...⟧ __)__
