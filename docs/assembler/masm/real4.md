# <a name="real4"></a>REAL4

Allocates and optionally initializes a single-precision (4-byte) floating-point number for each *initializer*.

## <a name="syntax"></a>構文

> *name*  **REAL4** *initializer* ⟦ __,__ *initializer* ...⟧
