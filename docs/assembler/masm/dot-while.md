# <a name="while-32-bit-masm"></a>.WHILE (32-bit MASM)

Generates code that executes the block of *statements* while *condition* remains true. (32-bit MASM only.)

## <a name="syntax"></a>構文

> **.WHILE** *condition*\
> *statements*\
> **.ENDW**
