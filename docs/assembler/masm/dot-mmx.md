# <a name="mmx-32-bit-masm"></a>.MMX (32-bit MASM)

Enables assembly of MMX or single-instruction, multiple data (SIMD) instructions. (32-bit MASM only.)

## <a name="syntax"></a>構文

> **.MMX**
