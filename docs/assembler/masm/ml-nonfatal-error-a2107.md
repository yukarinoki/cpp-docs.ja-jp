# <a name="ml-nonfatal-error-a2107"></a>ML の致命的でないエラー A2107

**暗黙的な far ジャンプまたは near ラベルへの呼び出しを含めることはできません**

暗黙的に離れた場所へのジャンプ、または別のセグメントの近くのラベルへの呼び出しが試行されました。
