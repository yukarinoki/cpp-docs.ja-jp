# <a name="ifidn-ifidni"></a>IFIDN、IFIDNI

テキスト項目が同一の場合は、アセンブリを行います。

## <a name="syntax"></a>構文

> **IFIDN**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Remarks

場合**IFIDNI**は、比較では大文字小文字を区別します。 参照してください[場合](../../assembler/masm/if-masm.md)の完全な構文です。
