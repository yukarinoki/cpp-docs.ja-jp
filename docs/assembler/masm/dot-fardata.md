# <a name="fardata-32-bit-masm"></a>.FARDATA (32-bit MASM)

When used with [.MODEL](../../assembler/masm/dot-model.md), starts a far data segment for initialized data (segment name FAR_DATA or *name*). (32-bit MASM only.)

## <a name="syntax"></a>構文

> **.FARDATA** ⟦*name*⟧
