# <a name="end-masm"></a>END (MASM)

モジュールの終了をマークし、必要に応じてプログラムのエントリポイントを*アドレス*に設定します。

## <a name="syntax"></a>構文

> **END** ⟦*address*⟧

## <a name="remarks"></a>Remarks

*アドレス*引数は32ビット MASM でのみ有効です。
