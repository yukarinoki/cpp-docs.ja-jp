# <a name="group-32-bit-masm"></a>GROUP (32-bit MASM)

Add the specified *segments* to the group called *name*. (32-bit MASM only.)

## <a name="syntax"></a>構文

> *name* **GROUP** *segment* ⟦ __,__ *segment* ...⟧
