# <a name="ifndef"></a>IFNDEF

Grants assembly if *name* has not been defined.

## <a name="syntax"></a>構文

> **IFNDEF** *name*

## <a name="remarks"></a>Remarks

See [IF](../../assembler/masm/if-masm.md) for complete syntax.
