# <a name="nolistif"></a>.NOLISTIF

条件が false (0) と評価される条件付きブロックの一覧を表示しません。

## <a name="syntax"></a>構文

> **.NOLISTIF**

## <a name="remarks"></a>コメント

既定値です。 と同じ[です.SFCOND](../../assembler/masm/dot-sfcond.md)。
