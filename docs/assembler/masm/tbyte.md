# <a name="tbyte"></a>TBYTE

Allocates and optionally initializes 10 bytes of storage for each *initializer*. Can also be used as a type specifier anywhere a type is legal.

## <a name="syntax"></a>構文

> ⟦*name*⟧ **TBYTE** *initializer* ⟦ __,__ *initializer* ...⟧
