# <a name="mktemp"></a>mktemp

この POSIX 関数は非推奨とされます。 代わりに ISO C++ 準拠の [_mktemp](mktemp-wmktemp.md) またはセキュリティが強化された [_mktemp_s](mktemp-s-wmktemp-s.md) を使用してください。
