# <a name="cdbpropidsetoperator-"></a>CDBPropIDSet::operator =
1 つのプロパティ ID のセットを別の ID プロパティ セットの内容を割り当てます。  
  
## <a name="syntax"></a>構文  
  
```cpp
      CDBPropIDSet& operator =(CDBPropIDSet& propset) throw();  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** atldbcli.h
