# <a name="compiler-error-c2234"></a>コンパイラ エラー C2234

'name': 参照の配列は無効です

参照へのポインターが許可されていないため、参照の配列のことはできません。

次の例では、C2234 が生成されます。

```
// C2234.cpp
int main() {
   int i = 0, j = 0, k = 0, l = 0;
   int &array[4] = {i,j,k,l};   // C2234
   int array2[4] = {i,j,k,l};   // OK
}
```
