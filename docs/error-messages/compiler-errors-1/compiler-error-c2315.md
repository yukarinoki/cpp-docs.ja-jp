# <a name="compiler-error-c2315"></a>コンパイラ エラー C2315

'type1': 参照は行番号の 'type2' でキャッチされました

例外の種類は、前にあるハンドラーによって処理されます。 2 番目の catch の参照は、最初の catch の型になります。
