# <a name="compiler-error-c2401"></a>コンパイラ エラー C2401

'identifier': 'context'' ではレジスタは基底レジスタでなければなりません

このコンテキストでは、間接メモリ オペランドに使用するレジスタは基底レジスタである必要があります。
