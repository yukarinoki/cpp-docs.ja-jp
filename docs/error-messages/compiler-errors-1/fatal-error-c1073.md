# <a name="fatal-error-c1073"></a>致命的なエラー C1073

インクリメンタル コンパイル (コンパイラ ファイル 'filename'、行番号) を伴う内部エラー

インクリメンタル コンパイルを使用せず、ファイルを再コンパイルします。
