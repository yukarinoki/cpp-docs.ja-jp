# <a name="compiler-error-c2334"></a>コンパイラ エラー C2334

unexpected token(s) preceding ': or {'; skipping apparent function body

次の例では、C2334 が生成されます。 このエラーは、エラー C2059 後に発生します。

```
// C2334.cpp
// compile with: /c
// C2059 expected
struct s1 {
   s1   {}   // C2334
   s1() {}   // OK
};
```
