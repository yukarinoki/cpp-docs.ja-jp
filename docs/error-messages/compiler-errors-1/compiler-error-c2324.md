# <a name="compiler-error-c2324"></a>コンパイラ エラー C2324
'identifier': 'name' の右側に予期しません。  
  
 正しくない識別子を使用して、デストラクターが呼び出されます。  
  
 次の例では、C2324 が生成されます。  
  
```  
// C2324.cpp  
class A {};  
typedef A* pA_t;  
int i;  
  
int main() {  
   pA_t * ppa = new pA_t;  
   ppa->~i;   // C2324  
   ppa->~pA_t();   // OK  
}  
```
