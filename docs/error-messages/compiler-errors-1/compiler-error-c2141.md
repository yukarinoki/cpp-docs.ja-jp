# <a name="compiler-error-c2141"></a>コンパイラ エラー C2141

配列サイズのオーバーフロー

配列は、2 GB の制限を超えています。 配列のサイズを小さきます。

## <a name="example"></a>例

次の例では、C2141 が生成されます。

```
// C2141.cpp
// processor: IPF
class A {
   short m_n;
};

int main()
{
   A* pA = (A*)(-1);
   pA = new A[0x8000000000000001];   // C2141

   A* pA2 = (A*)(-1);
   pA2 = new A[0x80000000000000F];   // OK
}
```
