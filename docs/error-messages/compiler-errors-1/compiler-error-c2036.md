# <a name="compiler-error-c2036"></a>コンパイラ エラー C2036

'identifier': サイズが不明です

操作`identifier`特定できない、データ オブジェクトのサイズが必要です。

## <a name="example"></a>例

次の例では、C2036 が生成されます。

```
// C2036.c
// a C program
struct A* pA;
struct B { int i; } *pB;
int main() {
   pA++;   // C2036, size of A not known
   ((char*)pA)++;   // OK

   pB++;   // OK
}
```

## <a name="example"></a>例

次の例では、C2036 が生成されます。

```
// C2036_2.cpp
// a C++ program
struct A* pA;
int main() {
   pA++;   // C2036, size of A not known
   ((char*&)pA)++;   // OK, if sizeof(A) == sizeof(char)
}
```
