# <a name="compiler-error-c2325"></a>コンパイラ エラー C2325

'type': 'name' の右側に予期しない型

型が正しくないデストラクター呼び出し。

次の例では、C2325 が生成されます。

```
// C2325.cpp
// compile with: /c
class A {};
typedef A* pA_t;
void f() {
    A** ppa = new A *;
    ppa->~A*;   // C2325

   pA_t *ppa2 = new pA_t;
   ppa2->~pA_t();   // OK
}
```
