# <a name="compiler-error-c2437"></a>コンパイラ エラー C2437

'identifier': 既に初期化されています

オブジェクトは、1 回だけ初期化できます。

次の例では、C2437 が生成されます。

```
// C2437.cpp
// compile with: /c
class A {
public:
   A(int i) {}
};

class B : A {
   B() : A(1), A(2) {}   // C2437
   B() : A(1) {}   // OK
};
```
