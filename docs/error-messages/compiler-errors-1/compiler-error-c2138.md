# <a name="compiler-error-c2138"></a>コンパイラ エラー C2138

すべてのメンバーを持たない列挙型を定義するには、無効です。

列挙体は、少なくとも 1 つのメンバーが必要と/Za (Microsoft の拡張機能を無効にする) が選択されています。
