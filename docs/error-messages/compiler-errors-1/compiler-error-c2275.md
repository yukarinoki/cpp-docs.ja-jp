# <a name="compiler-error-c2275"></a>コンパイラ エラー C2275

'identifier': この型の式としての無効な使用

式を使用して、`->`演算子と共に、`typedef`識別子。

次の例では、C2275 が生成されます。

```
// C2275.cpp
typedef struct S {
    int mem;
} *S_t;
void func1( int *parm );
void func2() {
    func1( &S_t->mem );   // C2275, S_t is a typedef
}
```
