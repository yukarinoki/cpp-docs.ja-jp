# <a name="compiler-error-c2177"></a>コンパイラ エラー C2177
定数が大きすぎます。  
  
 割り当てられている変数の型の定数値が大きすぎます。  
  
 次の例では C2177 が生成されます。  
  
```  
// C2177.cpp  
int main() {  
   int a=18446744073709551616;   // C2177  
   int b=18446744073709551615;   // OK  
}  
```
