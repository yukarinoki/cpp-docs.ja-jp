# <a name="compiler-error-c2446"></a>コンパイラ エラー C2446

'operator': 'type1' から 'type2' への変換はありません

コンパイラは変換できません`type1`に`type2`します。 C と C++ のセマンティクスに違反しているため、変換が意味、ありません。
