# <a name="compiler-error-c2077"></a>コンパイラ エラー C2077
非スカラー フィールド初期化子 'identifier'  
  
 ビット フィールドで、非スカラー値 (構造体、共用体、配列、またはクラス) を初期化しようとするとします。 整数または浮動小数点値を使用します。
