# <a name="compiler-error-c2100"></a>コンパイラ エラー C2100

無効な間接参照

間接演算子 ( `*` ) nonpointer 値に適用されます。

次の例では、C2100 が生成されます。

```
// C2100.cpp
int main() {
   int r = 0, *s = 0;
   s = &r;
   *r = 200;   // C2100
   *s = 200;   // OK
}
```
