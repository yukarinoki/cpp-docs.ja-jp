# <a name="compiler-error-c2052"></a>コンパイラ エラー C2052

'type': 無効な型の case 式

Case 式は、整数の定数である必要があります。

次の例では、C2052 が生成されます。

```
// C2052.cpp
int main() {
   int index = 0;
   switch (index) {
      case 1:
         return 24;
      case 1.0:   // C2052
      // try the following line instead
      // case 2:
         return 23;
   }
}
```
