# <a name="compiler-error-c3157"></a>コンパイラ エラー C3157
ParamArray 属性は、最後のパラメーターにのみ適用できます。  
  
 [ParamArray](https://msdn.microsoft.com/library/system.paramarrayattribute.aspx)属性が間違ったパラメーターに適用されました。
