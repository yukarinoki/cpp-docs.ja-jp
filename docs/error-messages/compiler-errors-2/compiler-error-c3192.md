# <a name="compiler-error-c3192"></a>コンパイラ エラー C3192

構文エラー: '^' プレフィックス演算子ではありません (候補' *' ですか?)

ハンドルは、逆参照演算子として使用できません。

次の例では、C3192 が生成されます。

```
// C3192.cpp
// compile with: /clr
using namespace System;

ref class MyClass {
public:
   MyClass () {}
   MyClass(MyClass%) {}
};

int main() {
   MyClass ^ s = gcnew MyClass;
   MyClass b = ^s;   // C3192

   // OK
   MyClass b2 = *s;
}
```
