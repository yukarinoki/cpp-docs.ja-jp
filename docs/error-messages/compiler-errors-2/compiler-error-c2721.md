# <a name="compiler-error-c2721"></a>コンパイラ エラー C2721
'specifier': 演算子のキーワードと型の間に無効なストレージ クラス指定子  
  
 型変換で、ストレージ クラスを指定することはできませんので、ユーザー定義型の変換は、すべての記憶域クラスに適用されます。
