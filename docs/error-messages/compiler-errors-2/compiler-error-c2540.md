# <a name="compiler-error-c2540"></a>コンパイラ エラー C2540

配列の範囲として非定数式

配列には、定数がバインドされている必要があります。

次の例では、C2540 が生成されます。

```
// C2540.cpp
void func(int n, int pC[]) {
   int i = ((int [n])pC)[1];   // C2540
}

void func2(int n, int pC[]) {
   int i = (pC)[1];   // OK
}

int main() {
   int pC[100];
   func(100, pC);
   func2(100, pC);
}
```
