# <a name="compiler-error-c2679"></a>コンパイラ エラー C2679

二項 'operator': 'type' 型の右辺のオペランドを扱う演算子が見つかりません (または、変換はありません)

この演算子を使うには、型を指定してこの演算子をオーバーロードするか、この演算子が定義された型への変換を定義する必要があります。

次の例では、C2679 が生成されます。

```
// C2679.cpp
class C {
public:
   C();   // no constructor with an int argument
} c;

class D {
public:
   D(int) {}
   D(){}
} d;

int main() {
   c = 10;   // C2679
   d = 10;   // OK
}
```
