# <a name="compiler-error-c2948"></a>コンパイラ エラー C2948
明示的なインスタンス生成。ストレージ クラス指定子 'specifier' は特殊化では使用できません。  
  
 ストレージ クラス指定子を使用することはできません (など`extern`) で既に明示的にインスタンス化されているテンプレート クラスの特殊化です。
