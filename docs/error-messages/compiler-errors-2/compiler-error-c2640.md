# <a name="compiler-error-c2640"></a>コンパイラ エラー C2640
'identifier': _based 修飾子の参照  
  
 `__based`修飾子はポインターのみで使用できます。  
  
 次の例では、C2640 が生成されます。  
  
```  
// C2640.cpp  
void f(int i) {  
    void *vp;  
    int _based(vp) &vr = I;  // C2640  
}  
```
