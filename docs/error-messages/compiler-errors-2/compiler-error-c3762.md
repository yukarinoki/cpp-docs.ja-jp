# <a name="compiler-error-c3762"></a>コンパイラ エラー C3762

属性 'attribute' を処理できません。

ユーザー定義の属性から継承する`System.Security.Permissions.SecurityAttribute`セキュリティ属性を定義に使用されています。 このような属性は、定義されている、同じアセンブリで使用できません。
