# <a name="compiler-error-c2702"></a>コンパイラ エラー C2702

_ _except は終了ブロック内では表示されない場合があります。

例外ハンドラー (`__try`/`__except`) 内にネストすることはできません、`__finally`ブロックします。

次の例では、C2702 が生成されます。

```
// C2702.cpp
// processor: x86 IPF
int Counter;
int main() {
   __try {}
   __finally {
      __try {}   // C2702
      __except( Counter ) {}   // C2702
   }
}
```
