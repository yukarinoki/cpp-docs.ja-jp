# <a name="compiler-error-c2582"></a>コンパイラ エラー C2582

'function' 関数は 'type' で使用できません。

代入演算子がないオブジェクトに割り当てるが試行されました。

次の例では、C2582 が生成されます。

```
// C2582.cpp
// compile with: /clr
using namespace System;

struct N {};
ref struct O {};
ref struct R {
   property O prop;   // C2582
   property O ^ prop2;   // OK
};

int main() {
   String ^ st1 = gcnew String("");
   ^st1 = gcnew String("");   // C2582
   st1 = "xxx";   // OK
}
```
