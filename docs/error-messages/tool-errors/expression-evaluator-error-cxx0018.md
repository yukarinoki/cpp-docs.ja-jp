# <a name="expression-evaluator-error-cxx0018"></a>式エバリュエーター エラー CXX0018
レジスタ名が無効です。  
  
 指定した登録が存在しないか、表示されることはできません。  
  
 [ウォッチ] ウィンドウには、次のレジスタを表示できます。  
  
 AX SP DS IP  
  
 BX BP ES FL  
  
 CX SI SS GS  
  
 DX DI CS SS  
  
 EAX ESP DS EIP  
  
 EBX EBP ES EFL  
  
 ECX ESI FS  
  
 EDX EDI CS  
  
 このエラーは、can0018 と同じものと同じです。
