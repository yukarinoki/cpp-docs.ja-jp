# <a name="resource-compiler-fatal-error-rc1121"></a>リソース コンパイラの致命的なエラー RC1121

ファイルの読み取り I/O エラー

リソース コンパイラは、ファイルを読み取れませんでした。 ファイルを含むドライブが使用可能であり、ファイルが正しいことを確認してください。
