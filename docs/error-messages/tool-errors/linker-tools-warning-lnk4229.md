# <a name="linker-tools-warning-lnk4229"></a>リンカー ツールの警告 LNK4229

無効なディレクティブ/directive が見つかった。無視されます。

ディレクティブを使用して渡す、[コメント](../../preprocessor/comment-c-cpp.md)プラグマが無効です。 無視`/directive`します。
