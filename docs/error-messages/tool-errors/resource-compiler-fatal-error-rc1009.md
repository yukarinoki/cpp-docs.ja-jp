# <a name="resource-compiler-fatal-error-rc1009"></a>リソース コンパイラの致命的なエラー RC1009

コンパイラの制限: マクロ 'macro' の入れ子のレベルが深すぎます

ファイルは、マクロの入れ子リソース コンパイラの制限を超えています。 そのマクロの入れ子の深さを小さく指定されたソース ファイルを変更します。
