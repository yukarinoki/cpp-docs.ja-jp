# <a name="resource-compiler-error-rc2171"></a>リソース コンパイラ エラー RC2171
不明な DIB ヘッダーの形式  
  
 ビットマップ ヘッダーではありません、 **BITMAPCOREHEADER**または**BITMAPINFOHEADER**構造体。
