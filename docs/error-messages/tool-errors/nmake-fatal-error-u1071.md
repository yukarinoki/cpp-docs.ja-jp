# <a name="nmake-fatal-error-u1071"></a>NMAKE の致命的なエラー U1071

ターゲット 'targetname' の依存関係ツリーに循環定義

指定したターゲットの依存関係ツリーに循環依存関係が存在します。 指定されたターゲットでは、指定されたターゲットの依存オブジェクトのいずれかの依存です。 循環依存関係が無効です。
