# <a name="nmake-warning-u4001"></a>NMAKE の警告 U4001
コマンド ファイルは、コマンドラインからのみ呼び出すことができます。  
  
 によって呼び出されるように、コマンド ファイルをアット マーク (**@**) 指定子は、別のコマンド ファイルの仕様を含めることはできません。 このような入れ子にすることはできません。 仕様が無視されました。
