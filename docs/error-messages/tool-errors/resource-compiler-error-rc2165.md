# <a name="resource-compiler-error-rc2165"></a>リソース コンパイラ エラー RC2165

文字列 DLGINCLUDE ステートメントでは見つかりませんでした。

ステートメントでは、有効なインクルード ファイルが指定されませんでした。

**DLGINCLUDE**ステートメントは、次の構文を使用する必要があります。

**DLGINCLUDE"**<em>filename</em>**.h"**
