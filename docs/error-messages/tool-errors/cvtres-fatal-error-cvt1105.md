# <a name="cvtres-fatal-error-cvt1105"></a>CVTRES の致命的なエラー CVT1105
ファイルにシークできません。  
  
 CVTRES は、ファイル内の場所に移動できませんでした。  
  
 このエラーは、ファイルの破損によって発生することができます。
