# <a name="resource-compiler-error-rc2163"></a>リソース コンパイラ エラー RC2163

[ASCII または VIRTKEY] に必要なアクセラレータの種類

`type` ACCELERATORS **ステートメントの** フィールドには、ASCII または VIRTKEY の値が必要です。
