# <a name="cvtres-warning-cvt4001"></a>CVTRES の警告 CVT4001
**コンピューターの種類が指定されていません。見なされます**   
 ***type***  
  
 CVTRES は、コンピューターの仕様を見つけられませんでした。 これには、指定したコンピューターの種類と見なされます。 場合は既定では、/MACHINE オプションを使用して CVTRES を正しくない、再実行してください。
