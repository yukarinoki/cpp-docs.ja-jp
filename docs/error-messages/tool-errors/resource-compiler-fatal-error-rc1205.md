# <a name="resource-compiler-fatal-error-rc1205"></a>リソース コンパイラの致命的なエラー RC1205

無効なコード ページ

コード ページの指定 (/c) オプションが、無効なコード ページの後にします。 参照してください[コード ページ](../../c-runtime-library/code-pages.md)で、*ランタイム ライブラリ リファレンス*詳細についてはします。
