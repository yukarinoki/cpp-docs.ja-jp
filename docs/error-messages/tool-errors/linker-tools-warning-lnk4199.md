# <a name="linker-tools-warning-lnk4199"></a>リンカー ツールの警告 LNK4199

/DELAYLOAD:dllname が無視されます。インポート dllname がありません。

無視`dllname`、関数のいずれかが必要がないためを`dllname`をエクスポートします。
