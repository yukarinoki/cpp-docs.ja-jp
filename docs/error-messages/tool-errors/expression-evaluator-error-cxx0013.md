# <a name="expression-evaluator-error-cxx0013"></a>式エバリュエーター エラー CXX0013
演算子がありません。  
  
 演算子は、式で必要でしたが、見つかりませんでした。 式の構文を確認します。  
  
 このエラーは、can0013 と同じものと同じです。
