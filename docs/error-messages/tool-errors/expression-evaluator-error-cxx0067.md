# <a name="expression-evaluator-error-cxx0067"></a>式エバリュエーター エラー CXX0067

関数の評価はサポートされていません

式には、関数呼び出しが含まれています。 式エバリュエーターは、関数呼び出しをサポートしていません。

このエラーは、can0067 と同じものと同じです。
