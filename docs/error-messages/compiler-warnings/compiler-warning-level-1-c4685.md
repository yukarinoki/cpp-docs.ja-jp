# <a name="compiler-warning-level-1-c4685"></a>コンパイラの警告 (レベル 1) C4685

テンプレート パラメーターの解析中に '> >' が必要ですが、'>>' が見つかりました。

テンプレートの定義が正しく終了しませんでした。
