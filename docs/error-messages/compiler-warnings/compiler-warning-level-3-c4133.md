# <a name="compiler-warning-level-3-c4133"></a>コンパイラの警告 (レベル 3) C4133
'type': 'type1' から 'type2' に互換性のない型  
  
 この警告は、さまざまな種類の 2 つのポインターを減算するしようとした可能性があります。  
  
 この警告を回避するには、適切な型キャストを提供します。
