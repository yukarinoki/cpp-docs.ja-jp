# <a name="compiler-warning-level-1-c4179"></a>コンパイラの警告 (レベル 1) C4179

> '//*': として解析 '/' および '/\*': 標準との混同 '//' のコメント  
  
__//\*__ 正しくないコメントの区切り記号。 使用して__//__ または__/ \*__ 代わりにします。
