# <a name="compiler-warning-level-1-c4613"></a>コンパイラの警告 (レベル 1) C4613

'segment': セグメントのクラスは変更できません

コンパイラによって使用されるセグメントと同じクラス名でセグメントを作成しようとしました。 新しいセグメント クラスは作成されませんでした。
