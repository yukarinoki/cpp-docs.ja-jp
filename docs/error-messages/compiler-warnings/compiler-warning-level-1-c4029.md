# <a name="compiler-warning-level-1-c4029"></a>コンパイラの警告 (レベル 1) C4029

宣言された仮パラメーター リストの定義

仮パラメーターの型、関数宣言では、関数定義と一致しません。 コンパイラは、定義のパラメーター リストを使用します。
