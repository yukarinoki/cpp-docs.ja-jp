# <a name="compiler-warning-level-1-c4410"></a>コンパイラの警告 (レベル 1) C4410

オペランドの無効なサイズ

命令のオペランドのいずれかが、サイズが正しくありません。 オペランドとして有効な最小のサイズが使用されました。
