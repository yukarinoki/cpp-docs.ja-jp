# <a name="compiler-warning-level-1-c4166"></a>コンパイラの警告 (レベル 1) C4166

**コンストラクターとデストラクターの呼び出し規約が無効です。**

コンストラクターとデストラクターは、明示的に **__clrcall**を指定する場合を除き、プラットフォームの既定以外の呼び出し規則を持つことはできません。
