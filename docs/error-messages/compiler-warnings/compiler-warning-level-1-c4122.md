# <a name="compiler-warning-level-1-c4122"></a>コンパイラの警告 (レベル 1) C4122

'function' : alloc_text プラグマが、extern c で宣言されている関数に対して作用しました

**alloc_text** プラグマは、 **extern c**で宣言されている関数にのみ適用されます。 これは、外部 C++ 関数には使用できません。

このプラグマは無視されます。
