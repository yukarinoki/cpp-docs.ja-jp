# <a name="compiler-warning-level-1-c4506"></a>コンパイラの警告 (レベル 1) C4506
インライン関数 'function' の定義がありません。  
  
 指定された関数が宣言され、インライン展開が定義されていないマークが付いています。  
  
 コンパイラは、関数インライン展開を行いました。  
  
 インライン化できませんを外部関数で宣言されていることを確認、`extern`キーワード。
