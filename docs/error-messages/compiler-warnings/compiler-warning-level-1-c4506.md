# <a name="compiler-warning-level-1-c4506"></a>コンパイラの警告 (レベル 1) C4506

インライン関数 'function' は定義されていません

指定された関数が宣言され、インライン展開が定義されていなかったマークが付いています。

コンパイラは、インラインではなく、関数を行いました。

インライン化を外部関数を宣言することを確認、`extern`キーワード。
