# <a name="compiler-warning-level-3-c4281"></a>コンパイラの警告 (レベル 3) C4281
型 'type' から 'operator ->' の再帰が発生しました  
  
 コードにより **-> 演算子**自体を呼び出す。  
  
 次の例では、C4281 が生成されます。  
  
```  
// C4281.cpp  
// compile with: /W3 /WX  
struct A;  
struct B;  
struct C;  
  
struct A  
{  
   int z;  
   B& operator->();  
};  
  
struct B  
{  
   C& operator->();  
};  
  
struct C  
{  
   A& operator->();  
};  
  
void f(A p)  
{  
   int i = p->z; // C4281  
}  
```
