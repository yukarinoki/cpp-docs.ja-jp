# <a name="hstringreferenceget-method"></a>HStringReference::Get メソッド
基になる HSTRING ハンドルの値を取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HSTRING Get() const throw()  
```  
  
## <a name="return-value"></a>戻り値  
 基になる HSTRING ハンドルの値。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
