# <a name="modulemodule-destructor"></a>Module::~Module デストラクター
モジュール クラスの現在のインスタンスの初期化を解除します。  
  
## <a name="syntax"></a>構文  
  
```  
virtual ~Module();  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** module.h  
  
 **名前空間:** Microsoft::WRL
