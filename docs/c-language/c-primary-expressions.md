# <a name="c-primary-expressions"></a>C 一次式
式のオペランドは "1 次式" と呼ばれます。  
  
## <a name="syntax"></a>構文  
 *primary-expression*:  
 *identifier*  
  
 *constant*  
  
 *string-literal*  
  
 **(**  *expression*  **)**  
  
 *expression*:  
 *assignment-expression*  
  
 *expression*  **,**  *assignment-expression*
